Resumen


En esta clase vimos una introducción al lenguaje HTML y CSS
También vimos como relacionarlos entre sí para poder realizar nuestra primera página web



Objetivo de la actual página


En esta actividad se busca la integración de los distintos conceptos abordados en la clase.

Algunos conceptos vistos:


- Como funciona una página web

- Que son y para que sirven los editores de texto

- Como aplicar estilos a un documento HTML utilizando CSS

- Los distintos rangos y jerarquías y como aplicarlos correctamente
